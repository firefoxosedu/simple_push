simple_push
=========

This is a FirefoxOS demo application to showcase the use of the push api included in versions 1.1 and above.

You can find more information about the Simple Push api used here in the following link:

https://developer.mozilla.org/en-US/docs/WebAPI/Simple_Push

In this app, we are using a external server side part that is documented in the following link:

https://github.com/arcturus/firefox-os-simplepush
